# Lekcja 5: Sygnały uprzywilejowane (stroboskop i flow)
Podstawowe ćwiczenie i kontynuacja diod programowalnych z kursu **Arduino cz. 2** z strony **Forbot**.

### Czego się nauczyłem:
* Wykorzystałem poznaną na ostatniej lekcji bibliotekę `Adafruit_NeoPixel.h` do stworzenia efektów świetlnych na pasku WS2812.
* Zrobiłem efekt przypominający światła policyjne (stroboskop).
* Zrobiłem efekt flow bardzo podobny do poprzedniego tylko z płynnym włączaniem diod.
* W zaprogramowaniu efektów pomogła mi funkcja `for()`, której nie będę omawiać ponieważ są to podstawy języka C które w jakimś stopniu umiem. (Planuje zrobić osobne repozytorium z nauką języka C)
